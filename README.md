### 💡 关于我

临床医学专业本科生，非计算机科班出身。
平时对 Agent 以及各种能让人“合法偷懒”的自动化有着强烈的兴趣。

在这个过程中，我和各类大模型形成了一套行之有效的工作流（确信）：

- 🙋‍♂️ **我**：描述需求
- 🤖 **AI**：一顿输出生成代码
- 🙋‍♂️ **我**：运行测试 ➡️ 满屏飘红报错
- 🤖 **AI**：淡定分析，给个补丁
- 🙋‍♂️ **我**：再次运行 ➡️ 继续报错
- 🤖 **AI**：再微调一下
- 🙋‍♂️ **我**：卧槽，怎么真跑起来了？！
