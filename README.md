关于我

临床医学专业本科生，不是计算机科班出身，也没有什么很硬核的开发背景。

只是对 AI 机器人这类东西比较感兴趣，尤其是那种不只会聊天，还能记点东西、调点工具、干点小活的 bot。

所以平时会写一些小插件、小工具，或者把现成的东西改成自己更顺手的样子。

准确来说，很多时候不是我在写代码，而是：

我：描述需求
AI：生成代码
我：运行报错
AI：修一下
我：再报错
AI：再修一下
我：怎么真跑起来了

一点想法

我觉得 AI Bot 最有意思的地方，不只是“能回复消息”。

更有意思的是，它能不能慢慢变成一个更顺手的工具：
能记住一点上下文，能在合适的时候调用工具，能帮忙处理一些重复的小事，也能在不把服务器炸掉的前提下干点活。

所以这个主页大概就是一个记录：

记录一个非专业选手，怎么靠兴趣、prompt、报错信息和一点点运气，把一些想法拼成能跑的东西。
