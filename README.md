### 💡 关于我

临床医学专业本科生，非计算机科班出身。
平时对 Agent（智能体）、插件开发以及各种能让人“合法偷懒”的自动化有着强烈的兴趣。

我的日常并不是“写代码”，而是“**凑**代码”。
在这个过程中，我和各类大模型（GPT、Claude、Gemini 等）形成了一套行之有效的工作流：

- 🙋‍♂️ **我**：描述需求
- 🤖 **AI**：一顿输出生成代码
- 🙋‍♂️ **我**：运行测试 ➡️ 满屏飘红报错
- 🤖 **AI**：淡定分析，给个补丁
- 🙋‍♂️ **我**：再次运行 ➡️ 继续报错
- 🤖 **AI**：再微调一下
- 🙋‍♂️ **我**：卧槽，怎么真跑起来了？！

*(P.S. 上面这段对话在我的日常里发生过无数次。)*

### 💭 一点想法

我觉得 Agent 最有意思的地方，不只是“能回复消息”，而是它能不能慢慢变成一个**专属的数字助理**：能记事、能调工具、能在不炸掉服务器的前提下帮我打杂。

所以这个主页就是个记录：记录一个连 Python 都看不懂的医学生，怎么靠着兴趣、Prompt、报错信息，以及一点点运气，把天马行空的想法变成能跑的代码。
