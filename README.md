### 💡 关于我

临床医学专业在读，没有硬核的计算机科班背景。

只是单纯对 AI 机器人这类东西有强烈的兴趣——尤其是那种不仅会聊天，还能记点东西、调点工具、干点小活的 bot。

所以我平时会写一些小插件，或者把现成的东西改成自己更顺手的样子。
准确来说，很多时候并不是我在“写”代码，我的真实开发日常其实是：

> **我**：描述需求
> **AI**：一顿输出生成代码
> **我**：运行测试 ➡️ 满屏飘红报错
> **AI**：淡定分析，给个补丁
> **我**：再次运行 ➡️ 继续报错
> **AI**：再微调一下
> **我**：卧槽，怎么真跑起来了？！

### 🛠️ 折腾方向

项目不一定很成熟，主要出发点是“自己用的时候遇到问题，就试着补一块”。

- **AI Bot / QQ Bot**：AstrBot 插件开发 / Agent 工具调用
- **环境与部署**：Docker / 远程服务器控制 / 沙箱执行
- **功能探索**：记忆与上下文管理 / 语音交互
- **实用主义**：一些能让人光明正大偷懒的自动化脚本

### 💭 一点想法

我觉得 AI Bot 最有意思的地方，不只是“能回复消息”。

更有意思的是，它能不能慢慢变成一个更顺手的工具：能记住一点上下文，能在合适的时候调用工具，能帮忙处理一些重复的小事，也能在不把服务器炸掉的前提下干点活。

所以这个主页大概就是一个记录：
记录一个非专业选手，怎么靠兴趣、Prompt、报错信息和一点点运气，把天马行空的想法，拼成能跑的东西。
